This fake web app simulates an Open Access integration to help prospective partners visualize how their content would integrate with Spotify. Click the Login/subscribe button to create an account and log in, so that you can see how the linking works.

You can decide if you want to lead users to your content on Spotify directly (and be redirected back here to link their account), or if you want users to link their account by clicking on the content.

Lunar Industries is a fictitious podcast network

Inspiring stories for your ambitious journey. Subscribe now to start listening.

Create an account & sign in

You can register with any email and password (it won't send any emails so make sure you remember what you enter here).

We used Firebase as an off-the-shelf solution since the purpose of this example app is not authentication itself. After signing in you can link to your Spotify account.
